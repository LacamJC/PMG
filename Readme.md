# Projeto Matematico Geral

## Sobre 
O projeto PMG é uma série de aplicaçoes matematicas colocadas em algoritmos para demonstrar seu funcionamento 

## Funcionalidades 
- Diferentes tipos de calculadoras
- Frequencia relativa absoluta
- Aumento salarial
- Calculo de emprestimo
 - Tabuada
 - Media ( em desenvolvimento)

 ## Como rodar este projeto

### Copie e entre no repositorio 
```bash 
    git clone https://github.com/LacamJC/PMG.git
    cd PMG
```

### Acesse pelo navegador o arquivo index.html e explore as funcionalidades !
